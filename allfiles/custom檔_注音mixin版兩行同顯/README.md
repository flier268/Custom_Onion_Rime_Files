# custom檔_注音mixin版兩行同顯

- 兩行（分段）同顯，一些 Rime 輸入法之輸入框無法支持「換行」，故不適用！

- 適用 Mac 鼠鬚管！💪

- 適用 Windows 新版小狼毫 0.17.4 版！🎉

- 不適用 Windows 舊版小狼毫！⚠️

- 不適用 Linux 下的 Rime 輸入法！⚠️

- 使用方法：將 .custom.yaml 檔案拉到上一層（與 .schema.yaml 同層），「重新部署」。

- 可用 lua 自動偵測系統並修正，實測重載狀況下會有延遲情形，影響輸入手感！故選擇手動添加。
